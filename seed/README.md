# Seed Template

This is the seed template used by Goddard to create new shadcn/ui projects.

## What's Included

- ✅ Vite + React + TypeScript
- ✅ Tailwind CSS v4 with Vite plugin
- ✅ shadcn/ui initialized with Neutral theme
- ✅ TypeScript path aliases configured (\`@/*\`)
- ✅ Button component from shadcn/ui
- ✅ Working example in App.tsx

## Customizing the Template

You can customize this template to match your preferred project setup:

1. Add more shadcn/ui components
2. Change the theme/colors
3. Add additional dependencies
4. Modify the default App.tsx
5. Add your own utilities or helpers

Any changes you make to this \`seed\` folder will be used for all new projects created by Goddard.

## Getting Started (Generated Projects)

After Goddard creates a project from this template:

\`\`\`bash
cd your-project-name
npm run dev
\`\`\`

Your app will be running at http://localhost:5173

## Adding More Components

\`\`\`bash
npx shadcn@latest add card
npx shadcn@latest add dialog
npx shadcn@latest add input
\`\`\`

Browse all components at: https://ui.shadcn.com/docs/components
